# BlogApp

A **web-based blog application** built with **ASP.NET Core** that demonstrates role-based authentication, CRUD operations, and deployment on IIS. This project is designed to showcase secure, full-featured blog management for both admins and users.

## Features

### Role-Based Functionality
- **Admin**
  - Create, edit, and delete posts
  - Moderate comments
- **Registered Users**
  - Comment on posts
- **Guests/Visitors**
  - Read posts and comments (cannot interact)

### Blog Management
- CRUD operations for posts
- Commenting system for registered users
- Posts and comments visible to all users

### Deployment
- Hosted on **IIS (localhost)** for development and testing
- Full end-to-end deployment experience

## Tech Stack
- **Backend:** ASP.NET Core  
- **Database:** Entity Framework  
- **Language:** C#  
- **Deployment:** IIS
